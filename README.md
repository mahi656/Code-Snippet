# Code Snippet Dashboard

A premium, high-performance workspace for code snippet management, version control, and productivity tracking.

---

## 🏛️ Core Pillars

### 1. Snippet Intelligence
The definitive way to manage your codebase modules.
- **Multi-layered Versioning**: Every save creates a restorable snapshot with change notes.
- **Advanced Search**: Debounced, backend-indexed search across titles, tags, and languages.
- **Lifecycle Management**: Integrated Trash system with secure recovery and permanent deletion.

### 2. Productivity & Milestones
Integrated tracking to keep your projects on schedule.
- **Interactive Calendar**: Map your coding milestones and project goals visually.
- **Milestone Workflows**: Directly link your development goals to specific code snippets.
- **Activity Monitoring**: Track contribution levels and frequencies across your workspace.

### 3. Unified Authentication
Secure, modern access for a seamless developer experience.
- **GitHub OAuth**: Full profile synchronization and single-click access.
- **QR Secure Login**: Fast, device-to-device authentication.
- **Protected Workspace**: JWT-encrypted session management.

## 🛠️ Technology Stack

- **Frontend**: Vite, React, Tailwind CSS, Lucide Icons
- **Backend**: Node.js, Express, TypeScript, Mongoose
- **Database**: MongoDB
- **Security**: JWT, GitHub OAuth, QR Encryption
